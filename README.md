# 생기부 분석기 (생기부 컨설팅 자료 생성 도구)

NEIS에서 출력한 **학교생활기록부Ⅱ PDF**를 넣으면, 브라우저에서 곧바로 분석 리포트를 만들어 주는 단일 HTML 도구입니다. 진로·상담 컨설팅, 정성평가 검토, 면접 준비에 쓰도록 설계했습니다.

## 핵심 원칙

- **100% 브라우저 처리 · 외부 전송 없음.** PDF는 서버로 올라가지 않고 브라우저 안에서만 파싱·분석됩니다. `pdf.js`까지 파일에 인라인되어 있어 인터넷 없이(`file://`) 열어도 동작합니다.
- **실측만, 판정 없음.** 지어낸 수치가 없으며 '우수/부족' 라벨을 붙이지 않습니다. 근거(원문 문장·빈도)와 함께 **신호만** 제시합니다.
- **개인정보 보호.** 가명 처리·강력 블라인드(이름·학교·대회명 마스킹) 옵션 제공. 임시 저장은 `sessionStorage`(로컬 전용, 탭 닫으면 삭제).

## 사용법

`생기부분석기.html`(또는 `index.html`)을 브라우저로 열고 생기부 PDF를 끌어다 놓으면 됩니다. 빌드 없이 그 파일 하나로 완결됩니다.

- 상단 **진로 핵심 키워드**에 학생 진로를 입력(계열 프리셋 버튼으로 시작 → 편집 → 적용/Enter).
- 리포트는 접이식 섹션(핵심 요약 / 활동·세특·역량 / 진로·전공 심화 / 교사용 진단 / 원자료)으로 정리됩니다.

## 분석 항목(요약)

- 성적 강점맵(교과군 등급 추이)·성취도 분포
- 입학사정관 3대 역량(학업·진로·공동체) 신호 + 역량 밸런스 레이더
- 진로 일관성·연계, 전공적합성 교과 융합 지도, 키워드 연결 네트워크
- 학년별 성장 궤적·활동 밀도 히트맵, 주도성 행위 동사, 탐구 서사(동기·과정·산출)
- 세특 정독 포인트 리뷰, 자동 추천 키워드·워드 클라우드·썬버스트
- (교사용) 검토 체크리스트, 아쉬운 점(Weak Spots), 다음 학기 액션 플랜, 면접·상담 예상 질문
- 내보내기: 인쇄(PDF), 키워드별 원문 문장 CSV, 다중 파일 일괄 요약 CSV, AI 프롬프트 복사

## 구조 (개발용)

파이프라인: `추출 → 파싱 → 분석 → 렌더`

| 파일 | 역할 |
|---|---|
| `extract.mjs` | pdf.js 좌표·괘선 기반 텍스트 추출(`extractRich`) |
| `parse.mjs` | 생기부Ⅱ 파서(구 9등급제 / 신 고교학점제 5등급제 양식 모두 지원) |
| `analyze.mjs` | 분석 엔진(전 지표·시각화 데이터) |
| `app_render.js` / `app_style.css` | 브라우저 렌더·스타일(공용) |
| `build_tool.mjs` | → `생기부분석기.html` 빌드(드래그&드롭 자립형 도구) |
| `build_report.mjs` | → `report.html` 빌드(프로토타입) |

```bash
npm install
node build_tool.mjs   # 생기부분석기.html 생성
```

## 지원 양식

- 구양식(9등급제, 표준편차 포함)
- 신양식(고교학점제 5등급제, 성취도별 분포비율)

텍스트 PDF만 지원합니다(스캔본·이미지 PDF는 인식 불가).
